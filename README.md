# Maquetación UI estilo WhatsApp — Entrega
## Proyecto web clon de Whatsapp

Este proyecto consiste en la maquetación en HTML y CSS de una interfaz tipo WhatsApp. La maqueta incluye un sidebar con lista de chats y un panel principal con la conversación. Solo se utilizaron HTML y CSS, sin JavaScript.

## Estructura del proyecto
*/index.html
/styles/main.css*

## Como visualizarlo
Abrir el archivo *index.html* en cualquier navegador web.

## Decisiones de diseño
- Se utilizaron colores aproximados a los de la interfaz original de WhatsApp.
- La tipografía fue elegida para asemejarse a la utilizada en la aplicación real.
- Se empleó **Flexbox** para la distribución del layout principal.
